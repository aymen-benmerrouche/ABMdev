# ABMdev - Software Developer Portfolio

A modern, professional glassmorphism-style portfolio website for Aymen Benmerrouche (ABMdev), showcasing expertise in web, mobile, desktop, and AI-powered application development.

## 🌟 Features

- **Glassmorphism Design**: Modern frosted glass aesthetic with blur effects and transparency
- **Dark Theme**: Elegant gradient background (dark blue → indigo → cyan)
- **Fully Responsive**: Mobile-first design that works on all devices
- **Smooth Animations**: Micro-interactions, hover effects, and scroll animations
- **SEO Optimized**: Semantic HTML5 structure for better search engine visibility
- **Performance Focused**: Clean, optimized code for fast loading

## 🎨 Design Elements

- **Typography**: Poppins for headings, JetBrains Mono for code elements
- **Color Palette**:
  - Primary: #0F172A
  - Accent: #38BDF8
  - Secondary Accent: #6366F1
  - Glass Cards: rgba(255,255,255,0.08)

## 📋 Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About Me**: Background, focus areas, and approach
3. **Education & Experience**: Timeline of academic and professional journey
4. **Skills**: Categorized technical and soft skills
5. **Projects**: Featured work including MFM Application, mobile apps, and web platforms
6. **Tasks & Responsibilities**: Key development activities
7. **Services**: Offered development services
8. **Contact**: Form and social media links

## 🚀 Quick Start

1. Clone or download this repository
2. Open `index.html` in your browser
3. No build process required - pure HTML, CSS, and JavaScript

2. Go to Settings → Pages
3. Select branch and root folder
4. Your site will be live at `https://[username].github.io/[repo-name]`

## 🎯 Customization

To customize this portfolio for your own use:

4. Update contact information and social links
